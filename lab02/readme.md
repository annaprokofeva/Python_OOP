# ЛР-2 — Коллекция объектов

## Цель работы

Научиться работать с коллекциями объектов, понять разницу между моделью сущности и контейнером объектов, реализовать собственный контейнерный класс с итерацией и базовыми операциями управления.

---

## Предметная область

**Образование**

Выбранная сущность: **`Student`** → контейнер: **`StudentGroup`**

---

## Описание класса

Реализован контейнерный класс `StudentGroup`, который хранит объекты `Student` и управляет ими.

---

## Реализованные возможности

### Базовые операции (оценка 3)
- `add(student)` — добавить студента с проверкой типа и дублей;
- `remove(student)` — удалить студента;
- `get_all()` — получить список всех студентов.

### Поиск и итерация (оценка 4)
- `find_by_name(name)` — поиск по имени;
- `find_by_age(age)` — поиск по возрасту;
- `find_by_course(course)` — поиск по курсу;
- `find_by_grade(min_grade, max_grade)` — поиск по диапазону оценок;
- `__len__()` — поддержка `len(group)`;
- `__iter__()` — поддержка `for student in group`.

### Расширенные операции (оценка 5)
- `__getitem__(index)` — индексация `group[0]`;
- `remove_at(index)` — удаление по индексу;
- `sort(key)` — универсальная сортировка по любому атрибуту;
- `sort_by_name()` — сортировка по имени;
- `sort_by_age()` — сортировка по возрасту;
- `sort_by_course()` — сортировка по курсу;
- `sort_by_grade()` — сортировка по оценке;
- `filter_active()` — только активные студенты;
- `filter_by_course(course)` — студенты заданного курса;
- `filter_scholarship()` — студенты, получающие стипендию.

---

## Защита коллекции

- нельзя добавить объект не являющийся `Student`;
- нельзя добавить дубликат (сравнение по имени, возрасту и курсу).

---

## Демонстрация работы

В файле `demo.py` показаны 10 сценариев:
- создание студентов и добавление в группу;
- удаление элемента;
- поиск по имени, курсу и оценке;
- итерация по коллекции и `len()`;
- проверка на дубликаты;
- индексация и удаление по индексу;
- сортировка по имени и оценке;
- фильтрация активных студентов и стипендиатов;
- срезы коллекции;
- сценарий перевода студентов на следующий курс.
